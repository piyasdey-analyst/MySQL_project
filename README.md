

# Pizza Sales Analysis Using MySQL
This project uses SQL to analyze pizza sales data and generate valuable insights into customer preferences, revenue trends, and overall business performance.

# Project Overview
The goal of this project is to analyze pizza sales data using SQL to better understand business performance and support informed decision-making.

# Business Questions Covered
1. Total number of orders placed
2. Total revenue generated
3. Best-selling pizzas by quantity
4. Highest revenue generating pizzas
5. Sales distribution by pizza category
6. Popular pizza sizes
7. Peak order hours and sales trends

This project proves that there’s a lot more to selling pizza than just a good recipe. By using MySQL to dig into the numbers, we’ve turned a mountain of raw sales data into a clear game plan for growth. Pinpointing exactly which pizzas people love and when they’re most likely to order takes the guesswork out of daily operations, making everything from staffing to stock management much more efficient. Ultimately, these insights give the business a competitive edge, ensuring every decision is backed by what the customers are actually buying.
