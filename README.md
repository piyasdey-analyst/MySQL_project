# MySQL_project

Pizza Sales Analysis using MySQL

This project analyzes pizza sales data using MySQL to uncover customer behavior, sales trends, and business performance insights.

The analysis includes:

* Total orders and total revenue
* Most ordered pizza sizes
* Top-selling pizza types
* Category-wise sales distribution
* Peak ordering hours
* Revenue contribution by pizza category
* Cumulative revenue trends over time

The project demonstrates the use of SQL queries, joins, aggregate functions, window functions, and business analysis techniques to turn raw sales data into meaningful insights.
